# Python Mastery Roadmap

A structured Python learning repository that starts from core syntax and grows into intermediate Python, advanced language features, backend-oriented development, interview preparation, and runnable mini-projects.

## What This Repository Contains

This repository contains 80 Python files:

- `01`-`14`: Core Python foundations
- `15`-`30`: Intermediate Python
- `31`-`45`: Advanced Python
- `46`-`60`: Backend-oriented Python
- `61`-`70`: Interview mastery and final revision
- `71`-`80`: Practical Python projects

Each lesson file is designed for step-by-step learning with explanations, examples, common mistakes, interview questions, and revision notes.

## Learning Phases

### Phase 1: Core Python

Topics include variables, data types, type conversion, input/output, operators, strings, lists, tuples, sets, dictionaries, conditionals, loops, functions, modules, and packages.

### Phase 2: Intermediate Python

Topics include exceptions, file handling, object-oriented programming, inheritance, polymorphism, encapsulation, abstraction, iterators, generators, decorators, context managers, regex, datetime, JSON, CSV, logging, collections, itertools, and functools.

### Phase 3: Advanced Python

Topics include comprehensions, lambda/map/filter/reduce, scope, namespaces, memory management, copying objects, magic methods, dataclasses, typing, enums, protocols, generics, metaclasses, descriptors, concurrency, and asyncio.

### Phase 4: Backend-Oriented Python

Topics include SQLite, MySQL, PostgreSQL, SQLAlchemy, requests, virtual environments, pip, testing, project structure, configuration, environment variables, CLI development, packaging, performance, and best practices.

### Phase 5: Interview Mastery

Topics include Python tricks, common errors, interview questions, coding patterns, time complexity, built-in functions, standard library, Python gotchas, backend examples, and final revision.

### Phase 6: Projects

The project phase includes runnable scripts that combine multiple Python concepts:

- `71_project_cli_todo.py`: CLI todo app
- `72_project_file_organizer.py`: File organizer
- `73_project_expense_tracker_sqlite.py`: SQLite expense tracker
- `74_project_password_manager.py`: Educational password manager
- `75_project_quiz_app.py`: Quiz app
- `76_project_api_client.py`: API client
- `77_project_blog_api_http_server.py`: Mini blog HTTP API
- `78_project_auth_system_sqlite.py`: SQLite auth system
- `79_project_ecommerce_backend.py`: Mini e-commerce backend
- `80_project_final_backend_app.py`: Final backend-style app

## How To Use

Run any lesson file with Python:

```bash
python 01_var.py
```

Run a project file:

```bash
python 71_project_cli_todo.py
```

Some project files create local demo data such as `.json` or `.db` files when executed.

## Recommended Study Path

1. Read one file at a time.
2. Run the examples.
3. Modify the examples.
4. Write your own version without looking.
5. Build the related project.
6. Revise common mistakes and interview questions.

## Requirements

Most files use only the Python standard library. A few lessons discuss third-party packages conceptually, but the core examples are designed to be beginner-friendly.

Recommended Python version:

```bash
python --version
```

Use Python 3.10 or newer.

## Goal

The goal of this repository is not only to learn Python syntax, but to become comfortable building real Python programs.
